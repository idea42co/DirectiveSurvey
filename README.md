#Directive Survey
An animated survey built with an Angular directive

##Usage
- Define all questions in the
```javascript
    $scope.set = [];
```
array and seperate them by commas
- All answers to the survey will be stored in the 
```javascript
    $scope.questions = {};
```
object along with the question
